发表于 2016-12-05   |   分类于 js

什么是offsetParent对象

为了方便后面的解释，先来了解一下offsetParent对象，它是是指元素最近的定位（relative,absolute）祖先元素，递归上溯，如果没有祖先元素是定位的话，会返回null

offsetLeft与offsetTop

offsetTop和offsetLeft表示该元素的左上角（边框外边缘）与已定位的父容器（offsetParent对象）左上角的距离。即：

  1. 如果存在offsetParent父元素，offsetTop是offsetParent父元素的padding + 本身元素的margin值
  2. 若没有，offsetTop则为本身的margin + 所有祖先元素的margin + padding + border值
  3. 如果存在offsetParent祖先元素，则offsetTop为offsetParent祖先元素的padding + 本身元素的margin + 其两者之间的层次元素的margin + padding + border值；offsetHeight与之相同

注：如果存在边距重叠的现象，也会影响到offsetTop，重叠部分取重叠后的最大值

offsetWidth、clientWidth、scrollWidth：

offsetWidth指绝对宽度 = 内容宽度 + padding 2 + scrollbar + border 2；

clientWidth内容可视区域 = 内容宽度 + padding * 2，不包括滚动条；

scrollWidth获取对象的滚动宽度，滚动内容的宽度 + 滚动内容的margin + 滚动内容的border 2 + 滚动内容的padding 2 +滚动内容父元素的padding；

scrollHeight获取对象的滚动宽度，滚动内容的宽度 + 滚动内容的margin 2 + 滚动内容的border 2 + 滚动内容的padding 2 +滚动内容父元素的padding 2；

offsetHeight、clientHeight、scrollHeight与上面算法相似

注：指定的width = 内容宽度 + scrollbar

clientLeft、scrollLeft与object.style.left

offsetleft返回的是数字，而object.style.left返回的是该object中内嵌样式style中left带px的字符串

scrollleft设置或获取的对象最左端和对象内容的最左端的距离；

clientleft是指获取对象的上边框的宽度只与本身的border有关；
